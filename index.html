QR Code Generator

Create a styled QR code with your own content, colours and logo.

How it works

  1. Select the QR code type.
  2. Enter the required details.
  3. Choose the QR code style and error correction.
  4. Adjust the main colour and gradient colour.
  5. Choose whether to include a logo.
  6. Adjust the logo size if needed.
  7. Press Generate QR Code.
